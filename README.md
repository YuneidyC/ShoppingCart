## Getting started

### Prerequisites

|               | Version       | Installation |
| ------------- | ------------- | ------------- |
| npm           | 11.6.1+       | `npm install npm@latest -g`  |

### Testing locally

```bash
git clone git@github.com:YuneidyC/ShoppingCart.git # Clone the repo
npm install # Install NPM packages
npm run start # Run the app
```
